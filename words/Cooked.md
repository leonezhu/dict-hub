Cooked 的多个意思 :: 
<!--ID: 1765120412773-->


**“cooked”** 这个词非常有意思，它的涵义在不同场景下变化很大，从字面到比喻甚至到俚语，跨度极大。我给你按层次拆开，方便你彻底“吃透”这个词。

---

# **1. 最基础：字面含义（literal meaning）**

### **cooked = 煮熟的 / 加工过的**

最直接的意思就是“经过加热烹饪过的”。

例句：

- _The chicken is cooked._  
    （鸡肉熟了。）
    
- _I prefer cooked vegetables._  
    （我喜欢煮熟的蔬菜。）
    

这里没有任何特殊意义，就是“加热完毕”。

---

# **2. 引申义：被处理过 / 不再原始**

cook 有“处理、加工、改变原始状态”的隐含逻辑，所以 **cooked = 不再原始，经过处理过**。

例句：

- _The books have been cooked._  
    （账目被做过手脚。 = 做假账）
    
- _cooked data_  
    （被人为修改过的数据）
    

这里的“煮熟”其实是隐喻：  
**原本天然的东西，被人为地“加工成另一种样子”。**

---

# **3. 俚语：糟糕了 / 完蛋了（slang）**

非常常见。

### **cooked = screwed / done for / finished**

意思是“你完了、你这下死定了”。

例句：

- _If the boss finds out, you’re cooked._  
    （老板知道的话，你完了。）
    
- _We’re cooked._  
    （我们死定了。）
    

逻辑：  
被“煮熟” = 没得救了（像鱼肉一样）。

---

# **4. 俚语新义（近几年特别流行）——“非常棒 / 炸裂 / 太强了”**

尤其美式网络文化里，“cooked”经常表示：

### **cooked = insane / fire / incredible（正向夸）**

例句：

- _This beat is cooked!_  
    （这段鼓点太炸裂了！）
    
- _Your idea is cooked!_  
    （你这个想法太牛了！）
    

这里的 metaphor 反过来了：**热到爆、热力十足 → 非常厉害**。

---

# **5. 情绪或身体状态：累坏了 / 被耗尽**

### **cooked = exhausted / burned out**

例句：

- _After working all day, I’m cooked._  
    （忙了一整天，我累趴了。）
    

和 burned out 接近，但更口语化。

---

# **总结：cooked 的五大语义层级**

|类别|含义|示例|
|---|---|---|
|1. 字面|煮熟了|cooked chicken|
|2. 引申|被处理/做假|cooked books|
|3. 负向俚语|完蛋了|you’re cooked|
|4. 正向俚语（很强）|炸裂、很牛|this beat is cooked|
|5. 状态|累坏了|I’m cooked|
