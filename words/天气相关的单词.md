# 🧩 **七、天气形容更自然的表达**

| 英文表达                      | 含义      |
| ------------------------- | ------- |
| It’s pouring.             | 雨下得很大。  |
| It’s drizzling.           | 正在下毛毛雨。 |
| It’s freezing outside.    | 外面冷得要命。 |
| The sun is blazing.       | 阳光很强烈。  |
| The sky is clear.         | 天空晴朗。   |
| It’s partly cloudy.       | 局部多云。   |
| It’s humid today.         | 今天很潮湿。  |
| There’s a chance of rain. | 有下雨的可能。 |
| The temperature dropped.  | 温度下降了。  |

# 🗣 **八、实用句型 Useful Weather Sentences**

**1）今天天气如何？**

* *How’s the weather today?*
* *What’s the weather like?*

**2）描述天气**

* *It’s sunny and warm today.*
* *It’s cold and windy outside.*

**3）预报天气**

* *It’s going to rain later.*
* *It might snow tomorrow.*
