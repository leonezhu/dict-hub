# English Learning Notes - 2026-04-13

> Real mistakes from today's Socratic dialogue with Echo. Grouped by type.

---

## 🗣️ Vocabulary / Word Choice Mistakes

### 1. "man learning use mean" → meant "what learning means"
- **What was said**: "I don't know what man learning use mean"
- **Correct**: "I don't know what learning means" / "I don't know the meaning of learning"
- **Rule**: Don't overcomplicate with extra nouns. "What X means" is the standard pattern.

### 2. "recover the orange setting" → meant "original setting"
- **What was said**: "Shall we recover the orange setting about the memory system?"
- **Correct**: "Should we restore the original setting for the memory system?"
- **Confusion**: Voice input confused "original" → "orange"
- **Note**: When speaking, pronounce "original" clearly: /ə-RIDJ-ə-nəl/

### 3. "radio" → meant "voice message"
- **What was said**: "I send a message, English message to you via radio"
- **Correct**: "I sent you an English voice message"
- **Rule**: "Radio" = broadcast medium. "Voice message" = what you send in chat apps.

## 📝 Grammar Mistakes

### 4. Missing subject-verb agreement
- **What was said**: "Maybe you should know what, how and why, and you also need know..."
- **Correct**: "...and you also need **to** know..."
- **Rule**: After "need", use infinitive "to + verb".

### 5. Double words (speech repetition)
- **What was said**: "I I thought it is useful to me"
- **Correct**: "I thought it was useful to me"
- **Also**: "have haven't enough time" → "haven't had enough time"
- **Note**: Normal in speech, just be aware when writing.

### 6. "it is just like received information" → passive voice needed
- **What was said**: "it's just like received information instead of learn"
- **Correct**: "it feels more like receiving information instead of learning"
- **Rule**: After "instead of", use -ing form: "instead of learning".

### 7. "speech a radio" → confused phrase
- **What was said**: "we can chat with a speech, speech a radio"
- **Correct**: "we can chat using voice messages"
- **Note**: Keep it simple. One clear phrase > multiple confused ones.

### 8. "I should our memory system have maybe have problem"
- **Correct**: "I think our memory system might have a problem"
- **Rules**:
  - "I should" → "I think" (opinion, not obligation)
  - "have maybe have" → pick one: "might have" or "maybe has"

### 9. "so you so absolute best we we I use we tried talk to you"
- **Correct**: "so you can absolutely see that I've been using WeChat to chat with you for about half an hour"
- **Note**: When speaking fast, words get scrambled. Slow down.

## 💡 Key Patterns

| Pattern | Mistake | Fix |
|---------|---------|-----|
| Need + verb | "need know" | "need **to** know" |
| Instead of | "instead of learn" | "instead of **learning**" |
| Opinion | "I should" (for guessing) | "I think" / "I guess" |
| Possibility | "have maybe have" | "might have" / "maybe has" |
| Extra nouns | "what man learning use mean" | "what learning means" |
| Time expressions | "half house" (voice) | "half an hour" |

---

## 🎯 Priority Practice Points

1. **Need to + verb** — most frequent mistake today
2. **Instead of + -ing** — use gerund after this phrase
3. **Simplify** — when unsure, use shorter sentences rather than complex ones
4. **Slow down** — many mistakes were from speaking too fast, not from not knowing
