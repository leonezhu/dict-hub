# Question (问题/疑问/质疑)

## 基础信息
**英文**：question [`ˈkwestʃən`]
**中文**：问题 / 疑问 / 质疑
**词性**：noun (名词) / verb (动词)
**词形变化**：questions (pl.), questioned (v.), questioning (v./adj.)

## 概念分析

### 1. 最一一对应的词
- **英语**：question → 问题/疑问/质疑
- **汉语**：问题 → question/problem/issue
- **特征**：英语1词→多概念，汉语多词→1概念

### 2. 一词多义 vs 一概念多词
- **英语**：question → 1词（名词+动词，多含义）
- **汉语**：问题 → 需要根据语境选择：
  - **提问**：question（疑问句）
  - **议题**：issue/problem（待解决）
  - **质疑**：challenge/doubt（怀疑）
  - **询问**：ask/inquire（动词）
- **分析**：英语通过词性转换，汉语通过不同词汇区分

### 3. 上下义关系
- **上义词**：inquiry（询问）, issue（议题）
- **下义词**：specific question（具体问题）, open-ended question（开放式问题）
- **汉语表达**：问题（上义）→ 疑问/质疑/议题（下义）

### 4. 同义词网络
- **英语同义词**：
  - 名词：inquiry, query, doubt, issue, problem
  - 动词：ask, inquire, challenge, doubt, interrogate
- **汉语近义词**：
  - 问题/疑问（中性提问）
  - 质疑/质询（怀疑挑战）
  - 讯问/盘问（正式审问）
- **角度差异**：
  - question：一般性提问
  - inquiry：正式调查
  - challenge：质疑权威
  - interrogate：审讯式询问

## 英汉对比

| 特征 | 英语 | 汉语 | 洞察 |
|------|------|------|------|
| **词性转换** | question（名+动） | 问题（名）/质疑（动） | 英语灵活，汉语明确 |
| **概念细分** | 1词涵盖多场景 | 多词区分不同语境 | 汉语更精确 |
| **语义范围** | 从提问到审问 | 词汇区分强度 | 英语语义更宽泛 |

## 实际应用

### 场景 1：课堂提问
- **英文**：Do you have any **questions**?
- **中文**：你有什么**问题**吗？
- **分析**：直接对应，最常用场景

### 场景 2：质疑观点
- **英文**：I **question** the validity of this approach.
- **中文**：我**质疑**这种方法的有效性。
- **分析**：动词用法，对应"质疑"而非"问题"

### 场景 3：正式调查
- **英文**：The committee will **question** the witness.
- **中文**：委员会将**询问**证人。
- **分析**：正式场合，对应"询问/盘问"

### 场景 4：基本疑问
- **英文**：That's a good **question**.
- **中文**：这是个好**问题**。
- **分析**：名词用法，直接对应

### 场景 5：挑战权威
- **英文**：Don't **question** my authority!
- **中文**：不要**质疑**我的权威！
- **分析**：强烈语气，对应"质疑"

### 场景 6：问卷调查
- **英文**：Please answer the **questionnaire**.
- **中文**：请回答**问卷**。
- **分析**：相关词汇，question + naire = 问卷

## 学习建议

### 记忆策略
- **词根记忆**：quest（寻求）+ ion = 寻求答案
- **词性转换**：question（名/动）→ 需要根据语境判断
- **场景记忆**：
  - 提问：ask a question
  - 质疑：question something
  - 问卷：questionnaire

### 使用注意
- **词性判断**：注意 question 作动词时的含义变化
- **搭配习惯**：
  - ask/raise/answer a question（提问/回答）
  - question something（质疑某事）
  - beyond question（毫无疑问）
- **强度区分**：question < challenge < interrogate

### 关联词汇
- **同根词**：questionnaire, questionable, unquestionable
- **近义词**：inquiry, issue, problem, doubt
- **反义词**：answer, solution, certainty
- **相关词**：ask, inquire, challenge, interrogate

## 深度洞察

### 1. 语言特征：词性灵活性的双刃剑
Question 的**一词多性**既是优势也是挑战：
- **优势**：简洁高效（"I question that" vs "I have doubts about that"）
- **挑战**：容易混淆（名词含义 vs 动词含义）
- **启示**：英语学习必须掌握词性转换

### 2. 概念映射：语境决定一切
同一个 question 在不同语境下：
- **中性**：What's your question?（你的问题是什么？）
- **怀疑**：I question his motives.（我质疑他的动机）
- **正式**：The suspect was questioned.（嫌疑人被审问）
- **汉语**：需要完全不同的词汇表达

### 3. 学习策略：语境敏感度培养
掌握 question 的关键是理解：
- **词性决定含义**：名词=问题，动词=质疑/询问
- **语境决定强度**：日常提问 vs 正式审问
- **文化差异**：西方鼓励 question 权威，东方相对含蓄

## 关键要点总结

### 翻译决策树
```
遇到 question 时：
├─ 是名词吗？
│  ├─ 日常提问？→ "问题"
│  ├─ 正式议题？→ "议题"或"问题"
│  └─ 疑问困惑？→ "疑问"
├─ 是动词吗？
│  ├─ 一般询问？→ "询问"或"问"
│  ├─ 怀疑挑战？→ "质疑"
│  └─ 正式审问？→ "盘问"或"讯问"
└─ 是形容词吗？→ "可疑的"（questionable）
```

### 常见错误避免
1. **词性混淆**：❌ "I have a question about this"（正确）vs ❌ "I question this"（含义不同）
2. **强度不当**：❌ "question the teacher"（日常）vs ✅ "ask the teacher"（更自然）
3. **语境误用**：❌ "question form"（不自然）vs ✅ "questionnaire form"（问卷表格）

### 记忆口诀
> **"Question 名动要分清，提问问题问老师，质疑挑战需谨慎，审问盘问正式场"**

### 核心概念
- **本质**：寻求答案或表达怀疑
- **英语特征**：词性灵活，语义宽泛
- **汉语特征**：概念精确，词汇多样
- **学习重点**：词性判断 + 语境敏感度
