# conveyed (传达 / 传递)

## 基础信息
**英文**：conveyed [kənˈveɪd]
**中文**：传达 / 传递 / 表达
**词性**：动词 (verb) - convey 的过去式和过去分词
**词源**：con- (共同) + vey (道路) → "带到一起" → 传达

## 概念分析

### 1. 一词多义 vs 一概念多词

**英语**：conveyed → 传达/传递/表达/运送/显示
- **信息传达**：convey information/meaning
- **情感表达**：convey emotions/feelings
- **物理运送**：convey goods (较少用)
- **显示表明**：convey an impression

**汉语**：传达 → 传递/转达/表达/表示/体现
- **传达**：正式、双向
- **传递**：物理或信息转移
- **表达**：情感、想法
- **转达**：间接传达

**分析**：英语 convey 一词多义，涵盖信息、情感、物理运输；汉语需根据语境选择不同词汇，概念更精细。

### 2. 上下义关系

**上义词**：communicate (沟通)
**下义词**：convey (具体传达方式)
**汉语表达**：传达 → 需要上下文确定具体方式

**关系链**：
- communicate (沟通) → convey (传达) → specific methods (具体方式)
- 汉语：沟通 → 传达 → 转达/表达/传递

### 3. 同义词网络

**英语同义词**：
- **communicate** - 更宽泛的沟通概念
- **transmit** - 强调传输过程（信号、信息）
- **express** - 强调表达情感想法
- **deliver** - 强调送达结果

**汉语近义词**：
- **传达** - 正式、双向
- **传递** - 物理/信息转移
- **表达** - 情感想法
- **转达** - 间接传达

**角度差异**：
- convey → 强调"带过去"的动作
- communicate → 强调"共同"的交流
- express → 强调"向外"的表现

## 英汉对比

| 特征 | 英语 | 汉语 | 洞察 |
|------|------|------|------|
| **概念细分** | convey 覆盖信息/情感/物理 | 需用不同词区分场景 | 英语概括性强，汉语场景化 |
| **表达习惯** | "convey + 宾语"直接搭配 | "传达/传递/表达 + 宾语" | 汉语动词更依赖上下文 |
| **翻译对应** | 1词→多场景 | 多词→1核心概念 | 翻译时需根据语境选择 |

## 实际应用

### 场景 1：信息传达
**英文**：The report **conveyed** the urgency of the situation.
**中文**：这份报告**传达**了情况的紧迫性。
**分析**：正式语境，"传达"最贴切，强调信息的正式传递。

### 场景 2：情感表达
**英文**：Her smile **conveyed** warmth and kindness.
**中文**：她的微笑**表达**了温暖和善意。
**分析**：情感语境，"表达"更自然，强调情感的外在表现。

### 场景 3：间接传递
**英文**：Please **convey** my thanks to the team.
**中文**：请向团队**转达**我的谢意。
**分析**：间接语境，"转达"强调中间人角色。

### 场景 4：含义显示
**英文**：His silence **conveyed** his disapproval.
**中文**：他的沉默**表明**了他的不赞成。
**分析**：暗示语境，"表明"或"显示"更贴切。

## 学习建议

### 记忆策略
- **词根记忆**：con- (共同) + vey (道路) = 带到一起 = 传达
- **场景记忆**：根据语境选择汉语对应词
- **对比记忆**：convey vs express vs transmit

### 使用注意
1. **正式程度**：convey 比 say/tell 更正式
2. **对象限制**：convey + 抽象概念（信息、情感、印象）
3. **时态变化**：convey → conveyed → conveyed

### 关联词汇
- **同根词**：conveyor (传送带), convoy (护送)
- **近义词**：communicate, transmit, express, deliver
- **反义词**：receive (接收), conceal (隐藏)

## 深度洞察

### 1. 语言特征
**英语**：convey 是"万能动词"，一个词覆盖多个相关概念，依赖宾语确定具体含义。
**汉语**：传达类词汇"分工明确"，不同场景用不同词，表达更精确但需要更多词汇储备。

**核心差异**：英语重"概括性"，汉语重"场景化"。

### 2. 概念映射
**映射类型**：一词多义 → 多词对应
**不对称性**：英语 convey = 汉语 传达+传递+表达+转达
**翻译决策树**：
```
conveyed 的语境是什么？
├── 正式信息传递 → 传达
├── 情感/想法 → 表达
├── 间接转告 → 转达
├── 物理运输 → 运送
└── 显示暗示 → 表明/显示
```

### 3. 学习策略
**掌握要点**：
1. **识别语境**：先判断是信息、情感还是暗示
2. **选择对应**：根据语境选择最贴切的汉语词
3. **积累搭配**：convey + information/emotions/meaning/feelings

**常见错误避免**：
- ❌ "convey a thing" (convey 不用于具体物体)
- ❌ "convey to him" (应说 convey to him that...)
- ✅ "convey information/emotions/meaning"

**记忆口诀**：
> "Convey 带到一起，传达信息和情意；
> 正式传达用传达，情感表达用表达；
> 间接转告要转达，显示暗示表明之。"

## 关键要点总结

### 核心概念
- **conveyed** = convey 的过去分词，表示"已被传达"
- **英语特征**：一词多义，概括性强
- **汉语特征**：多词分工，场景化精确

### 翻译决策树
1. **正式信息** → 传达
2. **情感想法** → 表达
3. **间接转告** → 转达
4. **显示暗示** → 表明/显示
5. **物理运输** → 运送

### 常见错误
- 混淆 convey 与 communicate 的正式程度
- 在具体物体上使用 convey
- 忽略语境选择错误对应词

### 记忆要点
- **词根**：con- + vey = 带到一起
- **场景**：信息→传达，情感→表达，间接→转达
- **搭配**：convey + 抽象概念
